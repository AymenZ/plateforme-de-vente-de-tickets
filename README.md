# Plateforme-de-vente-de-tickets
Ce projet consiste à développer une plateforme web de vente de tickets d’événements (concerts, films, théâtre, etc.). La plateforme permet aux clients d’acheter des tickets en ligne, aux organisateurs de publier et gérer leurs événements via un tableau de bord dédié, et aux administrateurs de superviser l’ensemble du système.
